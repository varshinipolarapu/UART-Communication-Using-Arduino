# UART Communication Using Arduino

## Description
This project demonstrates UART (Universal Asynchronous Receiver/Transmitter) communication using an Arduino Uno in Tinkercad. The Arduino receives data through the Serial Monitor and sends it back.

## Components Used
- Arduino Uno
- USB Serial Monitor
- Tinkercad Circuits

## Working
1. Initialize UART at 9600 baud rate.
2. Wait for serial data.
3. Read the received data.
4. Display the received data on the Serial Monitor.
